# Windows 安全模式网络状态说明

## 核心事实

| 模式 | 网络状态 | 说明 |
| --- | --- | --- |
| 安全模式 (Safe Mode) | ❌ 无网络 | 仅加载必要驱动，网络功能被禁用 |
| 带网络的安全模式 (Safe Mode with Networking) | ✅ 有网络 | 加载网络堆栈和基本驱动 |

## 详细说明

Windows 安全模式加载时：

- **默认行为**：只加载必需的驱动程序
- **网络状态**：网络适配器被禁用，大多数可选功能被关闭
- **驱动状态**：仅 essential drivers 被加载

### 安全模式 vs 带网络的安全模式

1. **普通安全模式 (Safe Mode)**
   - 无网络连接
   - 无网卡/网络适配器
   - 只能使用基础视频驱动（有时分辨率会降低）

2. **带网络的安全模式 (Safe Mode with Networking)**
   - 启用网络适配器
   - 可以访问网络/互联网
   - 适合需要下载驱动或工具的场景

## 常见场景

- **系统修复前**：如果需要下载驱动或工具，使用"带网络的安全模式"
- **查杀恶意软件**：普通安全模式已足够（多数恶意软件不在安全模式运行）
- **驱动问题排查**：普通安全模式可以排除驱动干扰

## 如何进入带网络的安全模式

### 方法一：msconfig

1. 按 `Win + R` 输入 `msconfig`
2. 选择"引导"选项卡
3. 勾选"安全引导" → 选择"网络"
4. 重启

### 方法二：设置

1. 设置 → 系统 → 恢复 → 高级启动
2. 疑难解答 → 高级选项 → 启动设置 → 重启
3. 按 `5` 或 `F5` 选择"带网络的安全模式"
