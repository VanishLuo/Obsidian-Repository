# Vim/GVim 常用命令

## 模式切换

| 命令 | 说明 |
|------|------|
| `i` | 在光标前插入 |
| `I` | 在行首插入 |
| `a` | 在光标后插入 |
| `A` | 在行尾插入 |
| `o` | 在下方新建行插入 |
| `O` | 在上方新建行插入 |
| `Esc` | 退出编辑模式 |
| `:` | 进入命令模式 |

## 保存与退出

| 命令 | 说明 |
|------|------|
| `:w` | 保存 |
| `:q` | 退出 |
| `:wq` 或 `:x` | 保存并退出 |
| `:q!` | 强制退出（不保存） |
| `:qa!` | 强制退出所有文件 |
| `:w !sudo tee %` | 以 sudo 权限保存只读文件 |

## 常用编辑命令

| 命令 | 说明 |
|------|------|
| `x` | 删除光标处字符 |
| `dd` | 删除当前行 |
| `yy` | 复制当前行 |
| `p` | 粘贴 |
| `u` | 撤销 |
| `Ctrl+r` | 重做 |
| `yy` | 复制整行 |
| `dd` | 剪切整行 |
| `cc` | 替换整行 |

## 光标移动

| 命令 | 说明 |
|------|------|
| `h/j/k/l` | 左/下/上/右移动 |
| `w` | 下一个单词词首 |
| `b` | 上一个单词词首 |
| `e` | 下一个单词词尾 |
| `0` | 行首 |
| `^` | 行首（非空白） |
| `$` | 行尾 |
| `gg` | 文件开头 |
| `G` | 文件末尾 |
| `:<n>` | 跳转到第 n 行 |
| `%` | 跳转到匹配括号 |

## 搜索与替换

| 命令 | 说明 |
|------|------|
| `/pattern` | 向下搜索 |
| `?pattern` | 向上搜索 |
| `n` | 下一个匹配 |
| `N` | 上一个匹配 |
| `:%s/old/new/g` | 全局替换 |
| `:%s/old/new/gc` | 全局替换（确认） |

## 窗口操作

| 命令 | 说明 |
|------|------|
| `:sp` | 水平分屏 |
| `:vsp` | 垂直分屏 |
| `Ctrl+w w` | 切换窗口 |
| `Ctrl+w h/j/k/l` | 切换到左/下/上/右窗口 |
| `:qa` | 关闭所有窗口 |

## 标签页操作

| 命令 | 说明 |
|------|------|
| `:tabnew` | 新建标签页 |
| `:tabn` 或 `gt` | 下一个标签页 |
| `:tabp` 或 `gT` | 上一个标签页 |
| `:tabclose` | 关闭标签页 |

## 其他常用

| 命令 | 说明 |
|------|------|
| `.` | 重复上次操作 |
| `~` | 大小写切换 |
| `>>` | 增加缩进 |
| `<<` | 减少缩进 |
| `==` | 自动格式化当前行 |
| `:%j` | 合并所有行 |
| `:set number` | 显示行号 |
| `:set nonumber` | 隐藏行号 |

## 外部命令

| 命令 | 说明 |
|------|------|
| `:!cmd` | 执行外部命令 |
| `:r !cmd` | 插入命令输出 |
| `:!sudo rm %` | 删除当前文件（需提前保存） |
